# ⚔️✨ Founder Quest - 起業家RPG診断アプリ

## 概要
起業家を目指す人・新規事業に関わる人が、40問のサーベイに回答することで、自分の起業家タイプをRPGキャラクターとして診断できるWebアプリ（MVP版）。

## 診断システム仕様

### 8パラメータ
| パラメータ | 表示名 |
|-----------|--------|
| VISION | ビジョン力 |
| ACTION | 実行力 |
| EMPATHY | 共感力 |
| ANALYSIS | 分析力 |
| CREATIVE | 創造力 |
| RISK | 挑戦力 |
| TEAM | 巻き込み力 |
| PERSIST | 継続力 |

### スコアリング
- **5段階評価（24問）**：1〜5点を対応パラメータに加算
- **シナリオ選択（8問）**：選択肢の対応パラメータに+2点
- **価値観選択（4問）**：選択肢の対応パラメータに+3点
- **自由記述（4問）**：スコア非対象（保存のみ）

### 8キャラクター
| キャラ | タイプ | 判定式 |
|--------|--------|--------|
| 勇者 | ビジョン牽引型 | VISION×1.5 + TEAM×1.0 + RISK×0.8 |
| 戦士 | 実行突破型 | ACTION×1.5 + PERSIST×1.0 + RISK×0.8 |
| 僧侶 | 共感支援型 | EMPATHY×1.5 + PERSIST×1.0 + TEAM×0.8 |
| 魔法使い | 創造発明型 | CREATIVE×1.5 + VISION×1.0 + ANALYSIS×0.8 |
| 盗賊 | 機会発見型 | RISK×1.5 + CREATIVE×1.0 + ACTION×0.8 |
| 商人 | 事業設計型 | ANALYSIS×1.5 + EMPATHY×1.0 + PERSIST×0.8 |
| 軍師 | 戦略分析型 | ANALYSIS×1.5 + VISION×1.0 + TEAM×0.8 |
| 召喚士 | チーム形成型 | TEAM×1.5 + EMPATHY×1.0 + VISION×0.8 |

- 最高スコア → **メインキャラ**
- 2番目 → **サブキャラ**
- 差が3点以内 → **ハイブリッドタイプ**表示
